# EdTech StartUp Analysis — Power BI Dashboard

## 📊 Overview
This project is a Power BI dashboard analyzing an EdTech course catalog to uncover key insights around course ratings, student enrollment, and category-wise distribution. The dataset was cleaned and transformed using Power Query before analysis.

## 🎯 Key Metrics
| Metric | Value |
|---|---|
| Total Number of Courses (after cleaning) | **3K** |
| Average Rating across all courses | **4.70** |
| Overall Students Registered | **8M** |
| Average Registrations per Course | **3.21K** |

## 🛠️ Tools Used
- **Power BI Desktop** — dashboard building and visualization
- **Power Query** — data cleaning and transformation
- **DAX** — custom measures and KPI calculations

## 📈 Dashboard Pages
- **Course Overview** — Total number of courses after data cleaning
- **Average Rating** — Average rating across all courses (post-cleaning)
- **Student Enrollment** — Total and average number of students registered per course
- **Distribution of Course Type Across Categories**
- **Category-wise breakdown**

## 🔍 Key Insight
With **3,000 courses** analyzed and an average rating of **4.70/5**, the dataset shows consistently high-quality course offerings. Courses collectively reached **8 million student registrations**, averaging **3,210 registrations per course** — indicating strong demand across the EdTech catalog.
